# Конфігурація ESP32 Radio

## Файл `config.h`

Цей файл містить **всі налаштування** проекту в одному місці. Ви можете змінити ці параметри без необхідності шукати їх по всьому коду.

### 📌 GPIO Піни

| Назва | Значення | Опис |
|-------|----------|------|
| `BTN_PWR_PIN` | 33 | Кнопка живлення |
| `BTN_LEFT_PIN` | 35 | Кнопка вліво |
| `BTN_RIGHT_PIN` | 39 | Кнопка вправо |
| `BTN_UP_PIN` | 32 | Кнопка вгору |
| `BTN_DOWN_PIN` | 36 | Кнопка вниз |
| `BTN_OK_PIN` | 34 | Кнопка OK |
| `ENC_A_PIN` | 13 | Енкодер сигнал A |
| `ENC_B_PIN` | 14 | Енкодер сигнал B |
| `ENC_BTN_PIN` | 19 | Енкодер кнопка |
| `IR_RECEIVER_PIN` | 15 | ІЧ приймач |
| `I2S_DOUT` | 27 | I2S data out |
| `I2S_BCLK` | 26 | I2S bit clock |
| `I2S_LRC` | 25 | I2S left/right clock |
| `TFT_DC` | 16 | Display Data/Command |
| `TFT_RST` | 17 | Display Reset |
| `TFT_MOSI` | 23 | SPI data |
| `TFT_SCLK` | 18 | SPI clock |
| `TFT_CS` | 5 | Chip select |
| `TFT_BL` | 4 | Backlight |

### ⏱️ Затримки та Таймаути

| Назва | Значення | Опис |
|-------|----------|------|
| `IR_COOLDOWN_MS` | 500 | Затримка після сигналу POWER ІЧ пульта |
| `IR_REPEAT_MS` | 200 | Затримка між повторними натисканнями ІЧ |
| `LONG_PRESS_MS` | 1000 | Час довгого натискання кнопок (1 сек) |
| `BTN_DEBOUNCE_MS` | 50 | Антидребез кнопок |
| `ENC_BTN_DEBOUNCE_MS` | 50 | Антидребез кнопки енкодера |
| `ENC_LONG_PRESS_MS` | 1000 | Довге натискання кнопки енкодера |
| `SOUND_SETTINGS_TIMEOUT_MS` | 10000 | Автозакриття налаштувань звуку (10 сек) |
| `LIST_AUTO_CLOSE_MS` | 10000 | Автозакриття списку станцій (10 сек) |
| `MESSAGE_TIMEOUT_MS` | 5000 | Час показу повідомлень (5 сек) |
| `OK_COOLDOWN_MS` | 300 | Затримка після LONG_PRESS кнопки OK |
| `WIFI_MAX_ATTEMPTS` | 20 | Максимальна кількість спроб підключення WiFi |
| `WIFI_RETRY_DELAY_MS` | 500 | Затримка між спробами підключення (500 мс) |

### 🎮 Коди ІЧ Пульта (RC-5)

| Назва | Код | Опис |
|-------|-----|------|
| `IR_BTN_POWER` | 0x0C | Живлення |
| `IR_BTN_VOL_UP` | 0x10 | Гучність + |
| `IR_BTN_VOL_DOWN` | 0x11 | Гучність - |
| `IR_BTN_MUTE` | 0x0D | Mute |
| `IR_BTN_CH_UP` | 0x20 | Канал + (попередня станція) |
| `IR_BTN_CH_DOWN` | 0x21 | Канал - (наступна станція) |
| `IR_BTN_PLAY_PAUSE` | 0x22 | Play/Pause |
| `IR_BTN_STOP` | 0x23 | Stop |
| `IR_BTN_INPUT` | 0x24 | Циклічна зміна входу |
| `IR_BTN_WIFI` | 0x2B | Вхід WiFi |
| `IR_BTN_COMPUTER` | 0x2C | Вхід Computer |
| `IR_BTN_TV_BOX` | 0x2D | Вхід TV Box |
| `IR_BTN_AUX` | 0x2E | Вхід AUX |

## 🔧 Як Використовувати

1. Відкрийте `src/config.h`
2. Змініть потрібні значення
3. Перекомпілюйте проект

**Увага!** Змінюйте значення обережно. Деякі параметри (як GPIO піни) залежать від вашої апаратної конфігурації.

Більше деталей про використання кожного параметра дивіться в коментарях до файлу `config.h`.
